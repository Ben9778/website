乡村会客厅

乡村会客厅是赤岸村把闲置的集体供销社进行活化利用，将其打造成集餐饮、民宿、展销、会议等多功能于一体的综合体。一层改造成乡村会客厅、美德银行与杂货铺，还预留了空间作为中草药或农耕博物馆，二层是乡村创客空间、培训基地和知青旅舍，后院则改造成人民公社大食堂。

赤岸的故事

赤岸村，位于永泰县丹云乡，是福州方言“七雁”的谐音，相传乡中有七座展翅俯冲、形态各异的雁形小山而得名。赤岸文化遗产资源相当丰富。2019年，赤岸村就被列入第五批中国传统村落名录，其中省级文物保护单位4处，历史建筑14处，文保单位和历史建筑总建筑面积达12881m²。

赤岸铳楼群

铳[chòng]楼，是具有防御功能的土堡群，建于清朝时期，现存扁店铳楼、前店铳楼、祥林店铳楼、则水店铳楼4座铳楼。放眼望去，100多年前的铳楼群矗立在溪旁，作为赤岸的时光标本，它见证了赤岸往昔的繁华。

水碓房酒吧

水碓房原是村民们舂米的地方，上个世纪八十年末颓废倒塌，2019年才被村里重新规划使用，他们保留了古老的石墙，复刻了缩小版的水车，对城里人来说，水车是陌生又稀罕的有趣物件，来赤岸游玩的人90%都会来看它。

邀三五好友汇聚与此，就着音乐，喝着小酒，身心俱欢。又或者临窗而坐，一面喝着小酒，一面欣赏着远处的田园风光，静静地享受着一个人独处的惬意时光，岂不快哉。
